# Threshold

**Threshold** é um motor/framework de jogo 2D construído em Java usando Swing e AWT. Ele fornece uma base para criar jogos baseados em tiles com gerenciamento de entidades, detecção de colisão e um loop de jogo básico.

## Índice da Documentação

A documentação para este projeto está organizada na pasta `docs/`.

- **[Arquitetura](docs/architecture.md)**: Visão geral da estrutura do projeto, componentes principais e relacionamentos de classes.
- **[Funcionalidade](docs/functionality.md)**: Explicação de como o motor de jogo funciona (Loop do Jogo, Mecânicas).
- **[Guia de Implementação](docs/implementation.md)**: Guia sobre como adicionar novas funcionalidades, entidades e mapas.
- **[Ideias](docs/ideas.md)**: Área de brainstorming para novas funcionalidades e mecânicas.

## Início Rápido

1. Abra o projeto na sua IDE Java.
2. Navegue até `src/Game.java`.
3. Execute o método `main` para iniciar a aplicação.
